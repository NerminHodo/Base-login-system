
# Base-login-system

Short description:  
Base login system  
**Enjoy!**

Technologies used:

-   React
-   Nodejs
-   MongoDB

To run it you should:

 - git clone https://github.com/NerminHodo/Base-login-system
 - open backend
 - npm install
 - npm start
 - open frontend
 - npm install
 - npm start
 - open browser and go to http://localhost:3000/
